# Sajid Khokhar - iOS Developer Portfolio

A stunning, premium, high-performance developer portfolio website designed for the Apple ecosystem. This site showcases projects, professional experience, education, and technical proficiency.

## Features

- **Premium Apple-Inspired Aesthetics**: Rich dark mode, subtle HSL gradients, elegant typography, glassmorphism headers, and smooth micro-interactions.
- **Ultra High-Performance**: 100% pure vanilla static website. Zero frameworks, zero compilers, zero dependencies, and zero build processes.
- **SEO & Accessibility Optimized**: Built using semantic HTML5 elements, correct heading structures, meta tags, and descriptive icon structures.
- **Fully Responsive**: Perfectly formatted across mobile, tablet, and desktop devices.
- **Scroll-Driven Interactions**: Clean Intersection Observer reveals elements dynamically and handles interactive skill bar fills when they enter the viewport.
- **Dynamic Elements**: Dynamically populated copyright year and active section navigation highlighting.
- **Self-Contained SVGs**: Embedded crisp vector icons for zero network latency and native styling control.

## Project Structure

```
├── index.html   # Main semantic page
├── style.css    # Clean vanilla CSS design system
├── script.js     # Responsive behavior, reveals, and toast notification JS logic
├── README.md    # Project documentation
└── .gitignore   # Git files ignore configuration
```

## Getting Started

Because this is a pure static site with no compiler steps or dependencies, launching it is instant:

1. **Open Directly**:
   Double click the `index.html` file to run the site directly in any web browser.

2. **Serve Locally**:
   Run a simple local HTTP server:
   - Using Python: `python3 -m http.server 8000` (Visit `http://localhost:8000`)
   - Using Node: `npx serve .` (Visit `http://localhost:3000`)
